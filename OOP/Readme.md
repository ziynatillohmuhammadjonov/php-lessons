1- dars OOP
OOP da class object property method tushunchalari bo'lib;
Class - ma'lum br xususiyat va methodlar jamlanmasi o'zi mustaqil tanaga ega emas;
Object - class dan xususiyat va metodlarni meros qilib olgan tanali jonzot
Property- class ichidagi xususiyatlar;
Method - class ichidagi funksiya va vazifalar clasni meros ilib olgan mavjudot bajaradigan

2-dars
OOP da asosiy 4 ta tamoyil mavjud: 
1. Encapsulation
2. Inheritance
3. Polymorphism
4. Abstraction
- Encapsulation tamoyiliga ko'ra klas ichidagi property ga to'g'ridan to'g'ri murojaat qilib bo'lmaydi. Unga murojaat klas ichidagi metodlardagina bo'ldi.

3-dars
- Inheritance - mavjud klassdan boshqa klass xususiyat va metodlarini meros qilib olishi. Bu orqali ummumiy xususiyat va metodlar qayta qayta klasslarda takrorlanishi oldina oladi. 
Magic method - bu class ichidagi o'zi bo'ladigan metodlar. Misol __constructor(), __invoke(), __toString(),... batafiz ```https://www.php.net/manual/en/language.oop5.magic.php```
